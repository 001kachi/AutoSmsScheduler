# AutoSmsScheduler

AutoSmsScheduler is a simple Android utility app designed to help users schedule SMS messages, calls, and reminders.  
This project is my first practical Android application and is being built as a learning-focused, real-world utility app.

The goal is not perfection, but steady improvement while working with core Android concepts.

---

## 📱 Platform
- Android (Android Studio)

## 🛠️ Tech Stack
- Kotlin
- Java
- Android SDK

---

## ✨ Current Features
- Schedule SMS messages
- Schedule calls
- Schedule reminders
- Send instant SMS messages
- Lock-screen UI for calls and reminders

---

## 🚧 Project Status
This project is **currently under active development**.

Some features are still being improved and refined, especially:
- Lock-screen call behavior
- Reminder UI handling
- Background task reliability

Updates will be pushed gradually as development continues.

---

## 🎯 Purpose of This Project
This project was created to:
- Practice Android development fundamentals
- Understand background tasks, alarms, and notifications
- Build a simple but practical utility app
- Serve as a junior developer portfolio project

---

## 🔐 Privacy & Security
- The app does not collect or transmit personal data
- All actions are performed locally on the device
- No external servers or APIs are used
